# Getting Started

Install `numeraire`, then run a complete backtest end to end on synthetic data — no external data
source required. The installation page covers the base spine, the companion-package extras, and a
development setup; the quickstart builds a point-in-time view, runs a walk-forward out-of-sample
backtest, evaluates it, and reads the tidy result rows.
